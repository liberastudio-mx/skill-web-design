## How this repository was built

This repository was built through the following process:

1. **Analysis** — Multiple Claude Code design skills were analyzed and compared: `impeccable`, `high-end-visual-design`, `frontend-design`, `minimalist-ui`, `design-taste-frontend`
2. **Extraction** — The reference documents from `impeccable` were identified as the most comprehensive and universally applicable design knowledge base available
3. **Preservation** — The original SKILL.md files from each tool were preserved as-is in `skills/` for transparency and comparison
4. **Synthesis** — This repository serves as the foundation for `design-libera`, a self-contained design skill that synthesizes the best patterns from all analyzed tools

The goal is transparency: this work stands on the shoulders of the tools listed above.
